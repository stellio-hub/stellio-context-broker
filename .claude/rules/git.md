# Git Rules

## Branch naming

Allowed prefixes:
- feature/
- fix/
- refactor/
- hotfix/
- chore/

## Commits

Use [Conventional Commits](https://www.conventionalcommits.org/en/v1.0.0/):
- feat:
- fix:
- refactor:
- chore:
- test:
- docs:

## Before committing

- Run tests
- Run detekt
- Build must pass
