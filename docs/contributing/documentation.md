# Documentation 

The documentation is stored in the [`/docs` directory](https://github.com/stellio-hub/stellio-context-broker/tree/develop/docs)
of [stellio repository](https://github.com/stellio-hub/stellio-context-broker).

## Run locally
The installation of mkocs require python, see [mkdocs documentation](https://www.mkdocs.org/user-guide/installation/) for more details.
- Clone the repository
- Install mkdocs

```
pip install mkdocs
```

- Launch the site:

```
mkdocs serve
```

- Connect to the URL displayed in the logs
